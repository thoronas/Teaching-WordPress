WordPress Content Types

WordPress comes with the following default types:

  • Post
  • Page
  • Attachment
  • Revisions
  • Nav Menus

But you can also define your own Custom Types.

Adding CPTs

CPTs are added using the register_post_type function.


add_action( 'init', 'create_post_type' );
  function create_post_type() {
    register_post_type( 'testimonial',
	  array(
	    'labels' => array(
	    'name' => __( 'Testimonials' ),	
	    'singular_name' => __( 'Testimonial' )
	  ),
	'public' => true,
	'has_archive' => true,
	)
  );
}

Different ways to create CPT

You can add the code directly in your functions.php.

Plugins: Both Custom Post Type UI and Stresslimit Custom Content Types are very useful.

Querying CPTs

Display the information by using WP_query


 $args = array( 'post_type' => 'testimonial', 'posts_per_page' => 10 );
 $loop = new WP_Query( $args );
   while ( $loop->have_posts() ) : $loop->the_post();
   the_title();
   the_content();
 endwhile;

Custom Taxonomies

Categories and Tags are examples of Taxonomies in WordPress. You have the ability to create your own.


add_action( 'init', 'create_custom_tax' );
					
function create_custom_tax() {
	$args = array(
		'label' => __( 'Genre' ),
		//hierarchical true is like categories
		'hierarchical' => true,
	);
	
	register_taxonomy( 'genre', 'post', $args);
}

This will create a custom taxonomy for posts called "Genre".

WordPress Codex on taxonomies

Taxonomy Templates

WordPress theme files supports Taxonomies.

  • taxonomy-{taxonomy}-{term}.php
  • taxonomy-{taxonomy}.php
  • taxonomy.php


Following our previous example our files would be called:

  • taxonomy-genre-fantasy.php
  • taxonomy-genre.php

Taxonomy Template Hierarchy

WordPress looks for the taxonomy template in this order:

  • taxonomy-{taxonomy}-{term}.php
  • taxonomy-{taxonomy}.php
  • taxonomy.php
  • archive.php
  • index.php

Using Taxonomies

Getting taxonomy terms in your templates using get_terms: 


 get_terms( $taxonomies, $args )

Getting taxonomy terms associated with a post using get_the_terms: 


get_the_terms( $id, $taxonomy )

get_terms example

Listing all the terms in our Genre taxonomy.

 
$terms = get_terms("genre"); 
if ( $terms && ! is_wp_error($terms) ){

	echo "<ul>";
	foreach ( $terms as $term ) {
		echo '<li><a href="'
		. get_term_link( $term->slug, $term->taxonomy ) 
		. '">' .$term->name. "</a></li>'; 	
	}
	echo "</ul>";
})

get_the_terms example

Listing all the terms in our Genre taxonomy related to an individual post.

 
$terms = get_the_terms($post->ID, "genre"); 
if ( $terms && ! is_wp_error($terms) ){

	echo "<ul>";
	foreach ( $terms as $term ) {
		echo '<li><a href="'
		. get_term_link( $term->slug, $term->taxonomy ) 
		. '">' .$term->name. "</a></li>'; 	
	}
	echo "</ul>";
})
