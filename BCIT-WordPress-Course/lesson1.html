WordPress

Introduction

What's WordPress?

WordPress is a Content Management System (CMS) that uses PHP and MySQL.

It's used to create dynamic websites instead of static ones.

Similarly to Joomla and Drupal, WordPress is distributed under a GPL license which makes it part of the open source platform.

WP Advantages

  • Can be used as a CMS & blog
  • Fully customizable
  • Thousands of themes and plugins to choose from
  • It's FREE
  • It's not as geeky as Drupal or Joomla
  • Releases come out regularly
  • The community is Awesome!

WP Disadvantages

  • Open to hackers
  • Releases come out regularly
  • Plugins are a necessary evil
  • It is still built by geeks
  • The learning curve can be steep

PHP/MySQL?

How much do you need to know

Compared to other platforms, WordPress is easy to use because you don't need to know any PHP, but you do need to be able to recognize PHP tags.

When we need to use PHP, we put it inside
<?php and ?> tags
so our server will know to process it as PHP.

WordPress
.com vs .org

WordPress.com

.COM == hosted solution.

Everything is taken care of for you - Hosting, server and database set up, spam, backups, upgrades.

The drawbacks of this option is that you must choose one of their themes, you can't install plugins and customization is very limited.

WordPress.org

.ORG == self hosted solution.

Web developers download the files, install them and run them on THEIR server.

Installing WordPress

To run WordPress you need a server.

Apache or Linux is best.

Options for building WordPress sites:

  1. Run a server locally on your machine
  2. Set up hosting with a hosting provider

Option 1 - Localhost

To develop your custom WordPress Theme on your computer, you'll need to run a "local server" which essentially mimicks a web server and includes
Apache, MySQL and PHP.

  • If you’re on a Windows computer download XAMP or Bitnami
  • If you’re running a Mac download MAMP

Option 2 - Hosting provider

If you want to share your website with the public, you'll need a domain name and a host.

Most hosting providers offer WordPress hosting these days, but do check to see if they provide:

  • PHP 5
  • MySQL 4.1.2 or greater
  • The mod_rewrite Apache module
